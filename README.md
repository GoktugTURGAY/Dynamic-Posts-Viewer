# Dynamic-Posts-Viewer
This project is a dynamic post pagination application built with HTML, CSS, and JavaScript. It fetches post data from the DummyJSON API using the XMLHttpRequest (XHR) object and displays the content with a fully interactive pagination system. The application includes loading states, error handling, page navigation controls, and accessible UI features to provide a smooth user experience. Posts are rendered dynamically through DOM manipulation, while pagination logic manages API requests efficiently using limit and skip parameters.
